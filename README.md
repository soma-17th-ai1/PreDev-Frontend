# PreDev-Frontend

Monogatari 엔진 기반 미연시 프론트엔드.

## 디렉토리 구조 및 역할

### `js/`
엔진 설정과 게임 스크립트 작성 및 로직을 담당하는 디렉토리.

*   **`script.js`** (핵심 스크립트)
    *   게임의 메인 스토리를 작성하는 파일. 대사, 선택지, 연출 등을 구현.
    *   `monogatari.script({ ... })` 내에 레이블(예: `"Start"`)을 키값으로 설정하여 씬(Scene)을 분리함.
    *   각 레이블의 값은 배열 형태로 구성되며, 명령어(`"show scene class"`), 대사(`"soma 안녕?"`), 선택지 객체 등을 순서대로 나열하여 스토리를 전개함.
*   **`main.js`**: 엔진 초기화, 게임 타이틀 및 버전 등 기본 정보 설정
*   **`options.js`**: 인게임 환경설정 기본값(언어, 텍스트 속도, 볼륨 등) 셋업
*   **`storage.js`**: 게임 내 저장/활용되는 전역 변수(호감도, 진행도 등) 초기 상태 정의

### `assets/` 
게임에 필요한 리소스 파일들을 관리함. (현재 주요 속성으로 캐릭터 및 배경 이미지만 표기)

*   `characters/`
    *   캐릭터 스프라이트 이미지를 저장.
    *   하위 폴더를 만들어서 그 폴더에 해당 캐릭터의 여러 스프라이트를 넣으면 됨
    *   `script.js`에서 에셋을 등록한 후, 스크립트 내에서 `"show character [캐릭터명] [표정]"` 포맷으로 호출.
*   `scenes/`
    *   스토리 배경으로 사용되는 이미지를 저장.
    *   Usage: 스크립트 내에서 `"show scene [배경명]"` 명령어로 배경 화면 전환.
